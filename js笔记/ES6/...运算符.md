#### 盒子的阴影

```css
  /*一边阴影*/
  .boxShadow {
    width: 100px;
    height: 100px;
    background-color: red;
    position: relative;
  }

  .boxShadow:after {
    content: '';
    width: 5px;
    height: 90px;
    position: absolute;
    left: 0px;
    top: 10px;
    background-color: green;
    box-shadow: 0px 0px 5px 5px green;
    z-index: -1;
  }
```

```css
  /*两边阴影*/
  .boxShadow{
    width: 100px;
    height: 100px;
    background-color: red;
    box-shadow: 5px 5px 5px green;
  }
```


```css
  /*三边阴影*/
  .boxShadow {
    width: 100px;

  }
```
