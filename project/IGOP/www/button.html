为防止button提交表单，需设置type="button"

如需div作为按钮容器，只需把button改为div，并去除type="button"属性即可.

一、基础按钮

    
对应的代码：
<button class="mg-btn mg-btn-primary">基础按钮</button>

二、置灰按钮

    
button实现代码：为防止button提交表单，需设置type="button"
<button class="mg-btn mg-btn-primary" type="button" disabled="disabled">置灰按钮</button>
div置灰按钮
div实现代码：需要配合js才能真正实现禁用功能
<div class="mg-btn mg-btn-primary" disabled="disabled">div置灰按钮</div> $("div[disabled='disabled']").click(function(){ return; }) 注意：$("div[disabled='disabled']")这里的选择器写法不唯一，可以是按钮的id或者class。

三、圆形按钮

对应的代码：
<button class="mg-btn mg-btn-primary mg-btn__circle"> <i class="iconfont icon-search01"></i> </button>

四、矩形按钮

    
对应的代码：
<button class="mg-btn mg-btn-primary mg-btn__rightAngle">矩形按钮</button> <button class="mg-btn mg-btn-default mg-btn__rightAngle">矩形按钮</button>

五、带有向右箭头按钮(纯css实现)

     
对应的代码：
<button class="mg-btn mg-btn-primary mg-btn__rightAngle btn-rightAngle-sm ">下一步 <span class="btn-arrow-right"></span> </button> <button class="mg-btn mg-btn-primary mg-btn__rightAngle btn-rightAngle-sm">上一步 <span class="btn-arrow-left"></span> </button>

六、带有向右箭头按钮(图片作为背景)

    
对应的代码：
<button class="mg-btn-arrow mg-btn__arrowRight">下一步</button> <button class="mg-btn-arrow mg-btn__arrowLeft">上一步</button>

七、ghost按钮

对应的代码：
<button class="mg-btn mg-btn__ghost">批量导入计费点</button>
